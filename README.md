# Be The Penguin Journey — Support

A short, silent journey told through symbolism — guide a penguin through a series of obstacles toward a distant mountain summit.

## Need help?

If you're experiencing an issue, have a question, or want to share feedback, please **[open an issue here](https://github.com/DevApp11/bethepenguin-support/issues/new)**. We'll respond as soon as possible.

*(Opening an issue requires a free GitHub account.)*

## In‑App Purchases

The game includes one optional consumable purchase ("Energy Shield"). If you have any issue with a purchase, open an issue above and include your order ID / receipt if possible. Purchases are handled entirely by Apple's App Store — refunds are requested through Apple at [reportaproblem.apple.com](https://reportaproblem.apple.com).
